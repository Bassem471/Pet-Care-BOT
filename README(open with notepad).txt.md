# &#x09;				Triger Words	

|**Function / Intent**|**Primary Trigger Words / Phrases**|**Requirement / Logic**|
|-|-|-|
|**Greeting**|**hi, hello**|**Triggered if the input contains basic greetings.**|
|**Update Pet Preference**|**favorite, pet**|**Must contain both words + a pet type: dog, cat, fish, or bird.**|
|**Update Category Preference**|**prefer, like**|**Must contain one of these + a category: diet, health, training, habitat, hygiene, or exercise.**|
|**Get Recommendation**|**recommend, suggest**|**Triggers the personalized engine based on saved preferences.**|
|**Ask Pet Question**|**feed, diet, train, sick, health, dog, fish, cat, bird**|**Triggered if the input contains any of these specific pet care keywords.**|
|**Get Summary \& History**|**summary, history**|**Triggers the analysis of total topics, mentioned pets, and current mood.**|
|**Bad Words Filter**|**kill, hurt, harm, abuse, poison, hit**|**If any of these are detected, the bot blocks the request for safety.**|
|**Repetition Detection**|**(Any repeated query)**|**Triggered automatically if the exact same question is asked more than once.**|
|**Exit Bot**|**exit, quit, bye**|**Closes the application loop.**|
